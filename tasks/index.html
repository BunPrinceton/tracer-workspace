Synapse Annotation

Learn to identify synapses using the four key indicators and classify the five synapse types.

Four Indicators of a Synapse

  1. T-bar (presynaptic density)
  2. Vesicle cloud
  3. Synaptic cleft
  4. Postsynaptic density (PSD)

Five Synapse Types

  • Chemical synapses (most common)
  • Gap junctions (electrical)
  • Polyadic synapses
  • Ribbon synapses
  • Neuromuscular junctions
