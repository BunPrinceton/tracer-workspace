Status: Deprecated

Omni

Status: Deprecated

3D proofreading and visualization tool for segmentation data. Replaced by WebKnossos and Neuroglancer.

Documentation

  • How to Omni Guide
  • Exporting from Omni
  • Omni Keyboard Commands (XLSX)

Glossary (Historical)

Term Definition
Dust Small voxel fragments inside large dendrites/axons
Merger AI error combining two non-connecting structures
Segment Collection of supervoxels
Supervoxel Smallest rearrangeable unit of segmentation
Spine Small membranous protrusion from dendrite

Eyewire

Status: Deprecated

Browser-based citizen science game for neuron reconstruction. Replaced by professional annotation tools.

Documentation

  • How to Eyewire (DOCX)

Key Features (Historical)

  • Gamified neuron tracing

  • Public participation model

Tool Evolution Timeline

  1. Eyewire Era - Citizen science approach
    Browser-based gamified neuron reconstruction for public participation.
  2. VAST + Omni Era - Professional desktop tools
    VAST for annotation, Omni for 3D proofreading. Ubuntu-based workflow.
  3. WebKnossos + Neuroglancer Era - Current
    Web-based tools with collaborative features, AI assistance, and CAVE integration.
