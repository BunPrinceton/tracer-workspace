Legacy Workflows

Desktop-Based Annotation

Deprecated
VAST + Omni Workflow

Original annotation workflow using desktop applications on Ubuntu workstations with network-mounted data storage.

Current: Web-based task workflow

Key Differences

  • Local data access vs. cloud-hosted volumes
  • Single-user vs. collaborative editing
  • File-based vs. database-backed annotations

Manual ID Tracking

Deprecated
Spreadsheet-Based

Manual tracking of segment IDs in spreadsheets without automated validation against changing segmentation.

Current: Automated ID validation via CAVE

Limitations

  • No detection of split/merge changes
  • IDs could become stale without notice
  • No supervoxel-based tracking

Historical Glossary

Terms from deprecated tools that may appear in legacy documentation. Some terms remain in use.

Omni Terminology
Dust Small voxel fragments erroneously included inside large dendrites or axons. Typically noise from segmentation.
Merger AI segmentation error where two non-connecting structures are incorrectly combined into one segment.
Split AI segmentation error where one continuous structure is incorrectly divided into multiple segments.
Segment A collection of supervoxels grouped together, representing a candidate neuron or fragment.
Supervoxel The smallest rearrangeable unit of segmentation. Groups of voxels that are always kept together.
Spine Small membranous protrusion from a dendrite, typically a postsynaptic structure.
VAST Terminology
Volume A 3D EM image stack loaded for annotation.
Brush Tool for painting segmentation labels onto voxels.
Label A semantic category assigned to segmented regions (e.g., "synapse", "mitochondria").
