GT Task Handling

SOP-001 — Version 1.0

Deprecated
Version 1.0
Effective Date 2025-06-01
Deprecated 2025-12-31
Owner GT Team Lead

Purpose

This Standard Operating Procedure establishes the basic workflow for handling Ground Truth (GT) annotation tasks.

Scope

This procedure applies to all annotators working on Ground Truth tasks.

Procedure

1. Task Assignment

  1. Check the task queue for assigned tasks
  2. Review task description before starting
  3. Confirm access to required datasets

2. Task Execution

  1. Open the task with correct viewport settings
  2. Follow the annotation protocol for the task type
  3. Save work frequently

3. Submission

  1. Mark task as "Complete"
  2. Add summary notes describing work completed

Reason for Deprecation

This version was superseded by v2.0 which includes:

  • WebKnossos-specific workflow steps
  • Cross-references to related SOPs
  • Mandatory quality check section
  • Updated submission procedure with new task status options
